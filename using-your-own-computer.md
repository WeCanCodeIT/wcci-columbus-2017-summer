# Using Your Own Computer

All of the software we use is open-source and/or free. You may use your own computer, but staff will not be able to guarantee assistance with this. If you are uncertain or run into difficulties, please use the laptop we have provided.

## Software List

- Java 8 Java Development Kit (JDK) (be sure to choose JDK, not JRE)
- Eclipse Java Developer Edition (NOT Java EE Edition)
	- In addition, install MoreUnit from Eclipse Marketplace (Help menu)
- Git client (with Git Bash on Windows machines)
- Sublime (or a programmer's text editor of your choice)
- Gradle
- lazybones (optional)
